1. ZAI_API_KEY
   الأداة: كل الـ chat + tool-calling + GLM models
   بتعمل: بتخلي الـ AI يرد على المستخدمين + يستدعي الأدوات
   اللينك: https://open.bigmodel.cn → سجل → API Keys → Create
   مجاني: ✅ 100% مجاني (GLM-4-Flash مجاني للأبد)
   البديل: ZHIPUAI_API_KEY (نفس الحاجة من نفس الموقع)

2. DATABASE_URL
   الأداة: قاعدة البيانات كلها (users, sessions, tools, skills, apps)
   بتعمل: بتخزين كل بيانات المنصة
   اللينك: https://supabase.com → سجل → New Project → خد connection string
   مجاني: ✅ (Supabase مجاني لحد 500MB)
   ملاحظة: إنت عندك DB مربوط — متغيرهش

3. NEXTAUTH_SECRET
   الأداة: NextAuth (Google OAuth + session security)
   بتعمل: بتشفير الـ JWT cookies — بدونها الـ sessions بتمسح كل restart
   اللينك: أي string عشوائي 32+ حرف
   مجاني: ✅ (مفيش موقع — اكتب أي حاجة عشوائية)

4. NEXTAUTH_URL
   الأداة: NextAuth redirect URLs
   بتعمل: بتحدد الـ domain بتاع المنصة
   القيمة: https://kopabdo-delta-ai-v2.hf.space
   مجاني: ✅

5. GOOGLE_CLIENT_ID + GOOGLE_CLIENT_SECRET
   الأداة: Google OAuth (Drive, Calendar, Tasks, Contacts, Docs, Sheets)
   بتعمل: بتخلي المستخدمين يربطوا Google accounts
   اللينك: https://console.cloud.google.com → APIs & Services → Credentials → Create OAuth client
   مجاني: ✅
   ملاحظة: لازم تضيف callback URL: https://kopabdo-delta-ai-v2.hf.space/api/auth/callback/google
