  Build: bun run build → next build (produces .next/standalone)

5.2 CRITICAL PATH
  page.tsx → auth-store.checkAuth() → AuthPage OR ChatApp
  ChatApp → ChatHeader + MessageList + ChatInput + StatusBar
  MessageList → MessageBubble (per message)
  ChatInput → /api/chat/stream (SSE streaming)

5.3 DATABASE
  Provider: PostgreSQL (Supabase in production)
  Fallback: SQLite (local dev via prisma/schema.prisma)
  Client: Prisma (singleton in src/lib/db.ts)
  Migrations: 6 migration files in prisma/migrations/

5.4 AUTH FLOW
  Login: /api/auth/login → JWT session token → localStorage (persisted)
  Verify: /api/auth/me → token rotation → updated localStorage
  Google: /api/auth/google → OAuth → /api/auth/google/callback
  Logout: /api/auth/logout → invalidate session → clear localStorage

5.5 STREAMING PIPELINE
  ChatInput → POST /api/chat/stream
  → auth check → rate limit → provider selection
  → ZAI SDK / HuggingFace / Cloudflare / Pollinations
  → SSE stream → MessageList → MessageBubble (typewriter effect)
  → Save to DB (conversation + messages)
