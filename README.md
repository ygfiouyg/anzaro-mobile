# Anzaro AI — The Smart Ball ⚽

منصة ذكاء اصطناعي عربي متكاملة مع الكرة الذكية والتحكم المنزلي.

## Features
- 🧠 **Personality AI** — 18-question onboarding quiz يبني profile شخصي
- 🎵 **Smart Ball Media Player** — راديو، قرآن، يوتيوب، TTS مع auto-play
- 🏠 **Home Assistant Control** — تحكم في الأجهزة والمشاهد بالعربي
- 📞 **Google Contacts Integration** — وصول موثوق لجهات الاتصال
- 📱 **PWA** — قابل للتثبيت على الموبايل بدون متجر
- 🔧 **341 MCP Tool** + 124 نموذج AI

## Tech Stack
- **Next.js 16** (App Router, webpack)
- **TypeScript 5**
- **Prisma ORM** (SQLite)
- **Tailwind CSS 4** + shadcn/ui
- **Zustand** + TanStack Query

## Configuration
This Space uses a Dockerfile to build and run the Next.js application on port 3000.
